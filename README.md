CSP451 CheckPoint 1

Description

- This repository contains my Week 1 CheckPoint 1 submission for CSP451 course. The main purpose of this project is to demonstrate my understanding that is based on version control concepts using both Git and GitHub. This project includes a simple website and also a proper Git workflows like commits, branching, and pull requests. In this assignment, I was available to determine and understand tracking changes, organization of files, and manage a repository with the use of Git.

Usage

- This project displays a basic webpage using HTML, CSS, and JavaScript.

1.) index.html - displays the main content

2.) style.css - provides basic styling

3.) script.js - outputs a message to the browser console

This repository is mainly used for learning and demonstrating Git version control workflows.
